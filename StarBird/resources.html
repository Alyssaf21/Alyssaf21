  • Github ⇨

    We used Github for sharing all the documents and the images for the webpage. You can find it on Github as "Space Birds", though the most up-to-date version is private.

  • ZyBooks ⇨

    We used Zybooks for the CSS styles and scrolling background animations from chapters two, three and four. We also used it to refer to some advanced HTML tags.

  • W3 Schools ⇨

    We used W3 schools for the navigation bar and for the CSS styles and animations. The website also helped us better understand HTML tags and syntax.

  • Piskel ⇨

    We used Piskel for drawing the characters and all the background art. Animations such as the bird's flapping and tumbling asteroids were also done here.

  • Website Repo⇨

    Originally a group project for CS301: Internet Applications Development, this is the first part of the project, which was to create a webpage to host the game.

  • Game Repo⇨

    The second half of the project entailed the creation of the game itself. This was done over the course of four days, though some group members continued to work on it long after.
